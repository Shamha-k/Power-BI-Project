 Omni Retail Customer Satisfaction & Loyalty Analysis

This project presents a descriptive analysis of customer satisfaction data using a Power BI dashboard developed for the DataDNA Onyx Data Challenge 2024. The analysis is based on a dataset of 120 customers and focuses on summarizing historical data to show what has occurred across different customer segments, how satisfaction levels vary, and how the data is distributed across demographic and geographic dimensions. The dataset includes variables such as age, gender, customer group, location, loyalty level, purchase history, satisfaction score, satisfaction factors, and support interaction, with the overall average satisfaction score recorded at 5.35 on a scale of 1 to 10.

The dashboard is structured to display the distribution of customers across categories including gender, group, age groups, loyalty levels, locations, purchase history, and support contact status. It also presents the frequency distribution of satisfaction scores, allowing clear visibility into how scores are spread across the dataset. In addition to counts and distributions, the analysis calculates average satisfaction scores across multiple dimensions, including customer groups, gender, age categories, cities, loyalty levels, satisfaction factors, and support interaction, enabling a direct comparison of values between segments.

Geographic representation is included through the use of location data, where customer distribution is visualized using latitude and longitude coordinates to show how data is spread across different cities. The dashboard consists of multiple interactive views that allow users to filter and explore the data dynamically, providing a consolidated view of customer satisfaction patterns without applying predictive models or assumptions.

This project strictly follows a descriptive analytics approach, focusing only on summarizing and presenting existing data. It does not include predictive, diagnostic, or prescriptive analysis. The purpose of the dashboard is to organize raw data into an accessible format, making it easier to observe distributions, compare aggregated values, and understand how customer satisfaction is represented across different segments within the dataset.

The dashboard was built using Microsoft Power BI, utilizing data modeling, aggregation functions such as averages and counts, and interactive filtering features to enable user-driven exploration. This project is intended for educational and portfolio purposes and demonstrates the application of descriptive analytics techniques in transforming structured data into an interactive visual report.
